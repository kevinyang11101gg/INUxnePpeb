# 项目简介

## 核心功能
- 垃圾图像数据采集、标注与数据增强
- 多类别垃圾分类数据集构建
- YOLOv8模型检测算法应用
- 迁移学习与超参数优化策略
- 图像上传、实时摄像头检测
- 结果标注与分类统计
- 模块化设计：数据预处理、YOLOv8推理、图形界面交互、结果输出

## 适用场景
- 垃圾分类数据集构建
- 垃圾识别与分析
- 实时垃圾检测与分类
- 环保监测与数据分析

## 技术特点
- 高效的C2f模块与动态检测头，平衡计算效率与检测精度
- mAP@0.5达94.2%的检测精度
- 60 FPS的单帧检测速度
- Python PyTorch框架后端，PyQt5前端构建
- 支持图片、视频和摄像头识别
- 清晰的模块化设计，易于扩展
